# MyRTOS 简介

欢迎阅读 MyRTOS 官方文档！本文档将帮助你全面了解 MyRTOS 的设计理念、核心特性和使用方法。

## 什么是 MyRTOS？

MyRTOS 是一款专为资源受限的嵌入式系统设计的**开源实时操作系统**。它具有以下特点：

- **硬实时调度**：基于优先级的抢占式调度，确定性响应
- **极小内存占用**：内核最小 2KB ROM / 512B RAM
- **模块化架构**：Kconfig 配置系统，按需裁剪
- **丰富生态**：网络协议栈、文件系统、Shell CLI 等组件
- **广泛平台支持**：ARM Cortex-M、RISC-V、ESP32 等

## 设计理念

### 1. 确定性优先

在实时系统中，**可预测性**比吞吐量更重要。MyRTOS 的所有内核操作都有确定性的时间复杂度，确保在最坏情况下仍能满足实时性要求。

### 2. 极简内核

内核只包含最核心的功能：任务调度、同步原语、通信机制、内存管理。所有其他功能以组件形式存在，可按需裁剪。

### 3. 开发者友好

- 清晰的 API 设计，命名规范统一
- 完善的错误检查和调试支持
- 丰富的示例代码和文档

## 系统架构

```
┌─────────────────────────────────────┐
│            应用层 (App)              │
├─────────────────────────────────────┤
│     组件层 (Components)             │
│  网络 │ 文件系统 │ Shell │ 日志     │
├─────────────────────────────────────┤
│         内核层 (Kernel)             │
│  任务 │ 信号量 │ 队列 │ 定时器 │ 内存│
├─────────────────────────────────────┤
│    驱动框架层 (Device Framework)     │
│  UART │ SPI │ I2C │ GPIO │ ...      │
├─────────────────────────────────────┤
│   板级支持包 (BSP) / HAL            │
│  STM32 │ ESP32 │ RISC-V │ ...       │
└─────────────────────────────────────┘
```

## 支持的硬件平台

| 架构 | 芯片系列 | 状态 |
|------|---------|------|
| ARM Cortex-M0/M0+ | STM32F0, NXP LPC11xx | ✅ 已支持 |
| ARM Cortex-M3 | STM32F1, GD32F1 | ✅ 已支持 |
| ARM Cortex-M4 | STM32F4, NXP RT10xx | ✅ 已支持 |
| ARM Cortex-M7 | STM32F7/H7 | ✅ 已支持 |
| ARM Cortex-A | i.MX6ULL | 🔧 开发中 |
| RISC-V | GD32VF103, CH32V | ✅ 已支持 |
| ESP32 | ESP32/S2/S3/C3 | ✅ 已支持 |
