## 关于 MyRTOS

MyRTOS 是一款面向嵌入式系统的轻量级实时操作系统，具有：

- ⚡ 硬实时调度，任务切换 < 1μs
- 🧠 极小内存，内核 < 2KB ROM
- 🔧 模块化架构，Kconfig 按需裁剪
- 🌐 丰富组件，网络/文件系统/Shell 开箱即用
- 🛡️ 安全可靠，MPU 保护 + 栈溢出检测
