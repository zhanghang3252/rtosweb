核心特性

为嵌入式场景深度优化的每一个细节，从内核到工具链，追求极致性能与可靠性。

⚡ 硬实时调度

MyRTOS 采用基于优先级的抢占式调度算法，确保高优先级任务在任何时刻都能获得 CPU 控制权。同时支持时间片轮转调度，为同优先级任务提供公平的 CPU 时间分配。

调度特性 说明 配置宏
抢占式调度 高优先级任务就绪时立即抢占当前任务 configUSE_PREEMPTION = 1
时间片轮转 同优先级任务按时间片交替执行 configUSE_TIME_SLICE = 1
优先级数量 支持 256 个优先级等级（0-255） configMAX_PRIORITIES = 256
任务切换时间 典型值 < 1μs（Cortex-M4 @ 168MHz） —
💡 提示：在 Cortex-M3/M4/M7 架构上，MyRTOS 利用 PendSV 异常实现零中断延迟的任务切换，上下文保存仅需 26 个寄存器压栈操作。

🧠 极小内存占用

MyRTOS 的内核代码最小可裁剪至 2KB 以下，RAM 占用低于 1KB。支持两种内存分配策略：

  • 静态分配：编译时确定所有资源，零运行时碎片，适合安全关键系统
  • 动态分配：内置 first-fit / best-fit 内存分配器，支持内存池，碎片率可控

RTOS 最小 ROM 最小 RAM 备注

🔧 模块化架构

MyRTOS 采用分层、模块化设计，内核、组件、驱动完全解耦。通过 Kconfig 配置系统，可按需裁剪功能模块，实现最小化构建。

内核组件清单

组件 说明 裁剪宏
任务管理 创建/删除/挂起/恢复任务 configUSE_TASK
信号量 二值信号量、计数信号量 configUSE_SEMAPHORE
互斥锁 优先级继承互斥锁 configUSE_MUTEX
消息队列 线程间消息传递 configUSE_QUEUE
软件定时器 回调式软件定时器 configUSE_TIMERS
事件标志组 多事件同步 configUSE_EVENT
内存池 固定块内存分配 configUSE_MEMPOOL

🔌 统一驱动框架

MyRTOS 提供统一的设备驱动模型（Device Framework），所有外设驱动遵循相同的接口规范，降低移植成本。

/* 设备驱动接口统一定义 */
struct device_ops {
    int (*init)(device_t *dev);         /* 初始化 */
    int (*open)(device_t *dev, int flags);  /* 打开 */
    int (*close)(device_t *dev);        /* 关闭 */
    int (*read)(device_t *dev, void *buf, size_t len);  /* 读 */
    int (*write)(device_t *dev, const void *buf, size_t len); /* 写 */
    int (*ioctl)(device_t *dev, int cmd, void *args);   /* 控制 */
};

🌐 网络协议栈

MyRTOS 内置轻量级 TCP/IP 协议栈（基于 lwIP），并提供上层应用协议支持，开箱即用的 IoT 连接方案：

  • TCP/UDP：完整的 Socket API，兼容 BSD Socket 编程模型
  • MQTT：内置 MQTT 3.1.1/5.0 客户端，支持 QoS 0/1/2
  • HTTP：HTTP/1.1 客户端与轻量级 Web 服务器
  • CoAP：RFC 7252 合规的 CoAP 实现
  • WiFi/BLE：ESP32/nRF 等平台的无线协议栈集成

🛡️ 安全与可靠性

面向工业、汽车、医疗等安全关键场景，MyRTOS 提供多层次的防护机制：

  • MPU 内存保护：利用硬件 MPU 隔离任务内存空间，防止非法访问
  • 栈溢出检测：编译时静态分析 + 运行时哨兵值检测
  • 看门狗监控：任务级看门狗，检测任务卡死
  • Trace 工具：系统运行时追踪，支持 Segger SystemView / Percepio Tracealyzer
  • 功能安全：通过 IEC 61508 SIL 3 认证（计划中）

🛠️ 开发工具链

MyRTOS 提供完善的开发工具链，提升开发效率：

  • Menuconfig：基于 Kconfig 的图形化配置工具，所见即所得
  • Env 工具：命令行环境管理，一键构建、烧录、调试
  • Studio IDE：基于 VS Code 的集成开发环境（计划中）
  • 包管理器：在线软件包仓库，一键集成第三方库
  • Shell CLI：交互式命令行，运行时调试与监控
