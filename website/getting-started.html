快速开始

从零开始，5 分钟将 MyRTOS 运行在你的开发板上。

📋 第一步：环境准备

MyRTOS 的构建系统基于 SCons，同时需要 ARM 交叉编译工具链。以下是在 Alibaba Cloud Linux 3 上的完整安装步骤：

1.1 安装基础依赖

# 更新系统包
sudo dnf update -y

# 安装基础开发工具
sudo dnf groupinstall -y "Development Tools"
sudo dnf install -y git python3 python3-pip scons

# 验证安装
python3 --version    # 需要 3.8+
scons --version      # 需要 4.0+

1.2 安装 ARM 工具链

# 方式一：从 ARM 官方下载（推荐）
wget https://developer.arm.com/-/media/Files/downloads/gnu/13.2.rel1/binrel/arm-gnu-toolchain-13.2.rel1-x86_64-arm-none-eabi.tar.xz
sudo tar -xf arm-gnu-toolchain-*.tar.xz -C /opt/
echo 'export PATH=/opt/arm-gnu-toolchain-13.2.Rel1-x86_64-arm-none-eabi/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# 方式二：使用 dnf 安装（版本可能较旧）
sudo dnf install -y arm-none-eabi-gcc arm-none-eabi-newlib

# 验证
arm-none-eabi-gcc --version

1.3 安装烧录工具

# OpenOCD（通用调试器）
sudo dnf install -y openocd

# 或者使用 J-Link（Segger 官方下载）
# https://www.segger.com/downloads/jlink/
💡 提示：如果你使用 ESP32 系列，需要额外安装 ESP-IDF 工具链。参考 ESP32 移植指南。

📥 第二步：获取源码

# 克隆 MyRTOS 源码仓库
git clone https://github.com/zhanghang3252/rtosweb.git
cd myrtos

# 查看目录结构
ls -la
# ├── kernel/         # 内核源码
# ├── arch/           # 架构相关代码
# ├── components/     # 组件（网络、文件系统等）
# ├── drivers/        # 设备驱动
# ├── boards/         # BSP 板级支持包
# ├── tools/          # 构建工具
# ├── examples/       # 示例代码
# ├── Kconfig         # 配置文件
# ├── SConstruct      # 顶层构建脚本
# └── README.md

⚙️ 第三步：配置工程

MyRTOS 使用 Kconfig 配置系统，支持图形化菜单配置和命令行配置两种方式。

3.1 图形化配置（推荐）

# 启动图形化配置界面
scons --menuconfig

# 界面操作说明：
# ↑↓    - 上下移动
# Enter - 进入子菜单 / 切换选项
# Space - 选中/取消选中
# Esc   - 返回上级菜单
# ?     - 查看帮助

3.2 关键配置项

配置项 说明 默认值
RT_BOARD 目标开发板型号 stm32f407-atk
RT_ARCH CPU 架构 arm
RT_TOOLCHAIN 编译工具链 gcc
configMAX_PRIORITIES 最大优先级数 32
configTOTAL_HEAP_SIZE 堆内存大小（字节） 10240

🔨 第四步：构建与烧录

4.1 编译固件

# 编译（自动检测 CPU 核心数并行编译）
scons

# 清除构建产物
scons --clean

# 指定编译线程数
scons -j8

4.2 烧录到开发板

# 方式一：使用 MyRTOS 内置烧录命令
scons --flash

# 方式二：使用 OpenOCD 手动烧录
openocd -f interface/stlink.cfg \
        -f target/stm32f4x.cfg \
        -c "program build/rtosweb.elf verify reset exit"

# 方式三：使用 J-Link 烧录
JLinkExe -device STM32F407VG -if SWD -speed 4000
# 在 J-Link 终端中执行：
# loadfile build/rtosweb.bin 0x08000000
# r
# g
# q

✅ 第五步：运行验证

# 连接串口终端（波特率 115200）
minicom -D /dev/ttyUSB0 -b 115200

# 或使用 screen
screen /dev/ttyUSB0 115200

# 预期输出：
# \ | /
# - MyRTOS -     Operating System
#  / | \     1.0.0 build May 25 2026
# 2006 - 2026 Copyright by MyRTOS Team
# msh >_
🎉 恭喜！如果看到以上输出，说明 MyRTOS 已成功运行在你的开发板上。接下来可以尝试 examples/ 目录下的示例项目。
