🔀 通过 Git 获取

推荐使用 Git 克隆仓库，方便后续更新和版本管理：

# 克隆最新稳定版
git clone -b v1.0.0 https://github.com/zhanghang3252/rtosweb.git

# 克隆开发版（最新特性，可能不稳定）
git clone -b develop https://github.com/zhanghang3252/rtosweb.git

# 后续更新
cd myrtos
git pull origin main
